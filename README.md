# Lebanese Flavors

## Description
Lebanese Flavors is a responsive restaurant website built using ReactJS.

The website showcases traditional Lebanese dishes, restaurant information, and a contact form.

## Features
- Home section
- Menu section
- About section
- Contact form
- Responsive design

## Technologies Used
- ReactJS
- CSS
- JavaScript

## Setup Instructions
1. Clone repository
2. Run npm install
3. Run npm.cmd start
